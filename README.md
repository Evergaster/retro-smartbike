# Retrofit Smart-Bike (UI Base)

Base de proyecto Android con Kotlin + Jetpack Compose para la app Retrofit Smart-Bike.

## Incluye

- Estructura tipo Clean Architecture + MVVM bajo `com.smartbike.retrofit`
- Tema Material 3 con paleta definida del mockup
- 9 pantallas Compose con placeholders visuales
- Navegacion Compose (`NavGraph`) y barra inferior reutilizable con FAB central
- ViewModels base por pantalla (sin logica de negocio real)

## Build rapido

```bash
cd "/home/evergaster/AndroidStudioProjects/MyApplication"
./gradlew :app:compileDebugKotlin
```
